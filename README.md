member
======

Og station member

How to build ?
======
mvn clean install

How to run ?
======
mvn jetty:run
 
API
======
* /api/error/400: handle 400 error
* /api/error/500: handle 500 error

* /api/integral(POST): add integral
* /api/integral(GET): retrieve integral
